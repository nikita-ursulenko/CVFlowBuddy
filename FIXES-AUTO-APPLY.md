# Исправления автоотправки CV

## 🐛 Найденные проблемы:

### 1. ❌ Хардкод лимита вакансий
**Проблема:** Независимо от настроек, всегда отправлялось только 10 CV

**Причина:** В `AgentControl.tsx` строка 228 было жёстко прописано `maxJobs: 10`

**Исправление:** Теперь используется значение из настроек `config.settings.maxCVDaily`

```typescript
// Было:
{ maxJobs: 10, minMatchScore: 70 }

// Стало:
{ 
  maxJobs: config.settings.maxCVDaily,  // Берёт из настроек (5, 10, 20, 50, 100)
  minMatchScore: 70,
  headless: config.settings.headless
}
```

### 2. ⏰ Таймаут слишком короткий
**Проблема:** При отправке 20 CV процесс прерывался через 2 минуты

**Причина:** API таймаут был 120000ms (2 минуты)

**Исправление:** Увеличен до 600000ms (10 минут)

```typescript
// Было:
signal: AbortSignal.timeout(120000) // 2 минуты

// Стало:  
signal: AbortSignal.timeout(600000) // 10 минут
```

### 3. 🔒 Браузер закрывался слишком быстро
**Проблема:** Браузер закрывался через 5 секунд после завершения, прерывая сохранение данных

**Причина:** `setTimeout(..., 5000)` в `agent-server-simple.js`

**Исправление:** Увеличено до 30 секунд

```javascript
// Было:
setTimeout(async () => { await agent.cleanup(); }, 5000); // 5 секунд

// Стало:
setTimeout(async () => { await agent.cleanup(); }, 30000); // 30 секунд
```

### 4. 📊 Улучшено логирование прогресса
**Проблема:** Не было видно сколько вакансий обработано

**Исправление:** Добавлено логирование в консоль:

```
======================================================================
📊 ПРОГРЕСС: 5 / 20 вакансий
   ✅ Отправлено: 3 | ⏭️ Пропущено: 2
======================================================================
```

### 5. ✅ Toast уведомления при сохранении
**Проблема:** Не было подтверждения что настройки сохранились

**Исправление:** Добавлен toast:

```typescript
toast.success(`✅ Настройки сохранены: ${formData.maxCVDaily} CV за запуск, интервал ${formData.intervalHours}ч`);
```

## ✅ Что теперь работает:

1. ✅ Настройка "20 за раз" применяется корректно
2. ✅ Настройки сохраняются в localStorage и не сбрасываются
3. ✅ Процесс обработки не прерывается досрочно
4. ✅ Видно прогресс обработки в реальном времени
5. ✅ Браузер не закрывается до завершения всех операций

## 🚀 Как проверить:

1. Откройте приложение: http://10.211.55.5:5004
2. Перейдите в раздел "Автоагенты"
3. Нажмите ⚙️ Настройки
4. Выберите "20 CV" в параметре "Максимум CV за один запуск"
5. Нажмите "Сохранить"
6. Должен появиться toast: "✅ Настройки сохранены: 20 CV за запуск..."
7. Запустите автоотправку
8. В консоли браузера (F12) будет виден прогресс

## 🎯 Следующие шаги:

Если проблемы сохраняются:

1. Откройте консоль браузера (F12)
2. Перейдите на вкладку "Network"
3. Запустите автоотправку
4. Проверьте запрос к `/api/agent/auto-apply-jobs`
5. Посмотрите в "Payload" параметр `maxJobs` - должен быть 20
6. Проверьте логи в терминале backend

Если агент "включается-выключается":
- Проверьте что sessionId сохранён
- Проверьте что cookies валидны (файл `lucru-cookies.json`)
- Посмотрите логи планировщика в консоли браузера
