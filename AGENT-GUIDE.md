# 🤖 Руководство по автоматическому агенту

## 📋 Обзор

Автоматический агент - это система, которая **автоматически отправляет ваше CV на IT вакансии** на сайте lucru.md по заданному расписанию.

## 🚀 Как запустить агента

### 1. Подготовка

1. **Загрузите CV файл** в разделе "CV"
2. **Войдите в аккаунт** lucru.md через кнопку "Войти в аккаунт"
3. **Настройте параметры** через кнопку "Настройки агента"

### 2. Настройки агента

#### ⏰ **Интервал проверки**

- **1 час** - проверяет каждые 60 минут
- **2 часа** - проверяет каждые 2 часа
- **3 часа** - проверяет каждые 3 часа
- **6 часов** - проверяет каждые 6 часов
- **12 часов** - проверяет каждые 12 часов
- **24 часа** - проверяет раз в день

#### 📊 **Максимум CV в день**

- **10 CV** - отправляет до 10 CV за день
- **20 CV** - отправляет до 20 CV за день
- **50 CV** - отправляет до 50 CV за день

#### 🕐 **Рабочие часы**

- **Начало работы** - с какого часа начинать (например, 09:00)
- **Конец работы** - до какого часа работать (например, 18:00)

#### 🔧 **Дополнительные настройки**

- **Автозапуск** - автоматически запускать агента при открытии приложения
- **Включить агента** - разрешить автоматическую работу
- **Фоновый режим** - работать без открытия браузера (рекомендуется)

### 3. Запуск агента

1. **Нажмите "Запустить агента"** - агент начнет работать по расписанию
2. **Статус отображается** в карточке агента:
   - 🟢 **Работает** - агент активен и отправляет CV
   - 🟡 **Приостановлен** - агент остановлен, но можно возобновить
   - 🔴 **Остановлен** - агент полностью остановлен

## 📈 Мониторинг работы

### Статистика в реальном времени

- **Отправлено CV** - общее количество отправленных CV
- **Ошибок** - количество ошибок при отправке
- **Всего обработано** - общее количество обработанных вакансий
- **Вакансий на сайте** - количество доступных IT вакансий

### Логи активности

- **Последние действия** - история отправки CV
- **Статус отправки** - успешно/ошибка/в процессе
- **Время выполнения** - когда была отправка

## ⚙️ Как это работает

### 1. **Планировщик задач**

- Агент запускается по расписанию (каждый N часов)
- Проверяет, что сейчас рабочие часы
- Запускает автоматическую отправку CV

### 2. **Автоматическая отправка**

- Переходит на страницу IT вакансий
- Находит все доступные вакансии
- Для каждой вакансии:
  - Проверяет наличие кнопки "Быстрая отправка CV"
  - Если есть - отправляет CV
  - Если нет - скрывает вакансию и записывает ошибку

### 3. **Фоновый режим**

- Работает без открытия браузера
- Не мешает работе за компьютером
- Автоматически закрывается после выполнения

## 🛠️ Управление агентом

### Кнопки управления

- **🟢 Запустить агента** - начать автоматическую работу
- **🟡 Пауза** - временно приостановить (можно возобновить)
- **🔴 Стоп** - полностью остановить агента
- **🔄 Возобновить** - продолжить работу после паузы

### Разовая отправка

- **⚡ Разовая отправка** - отправить CV прямо сейчас (не ждать расписания)
- Полезно для тестирования или срочной отправки

## 📊 Анализ результатов

### Успешные отправки

- CV отправлено на вакансию
- Записывается в статистику
- Отображается в логах

### Ошибки

- Вакансия без кнопки "Быстрая отправка CV"
- Проблемы с сетью
- Ошибки авторизации

### Скрытые вакансии

- Вакансии без возможности быстрой отправки
- Автоматически скрываются
- Записываются как ошибки

## 🔧 Решение проблем

### Агент не запускается

1. Проверьте, что сервер агента работает (зеленый статус "Онлайн")
2. Убедитесь, что вы авторизованы в lucru.md
3. Проверьте, что CV файл загружен

### Агент не отправляет CV

1. Проверьте настройки "Максимум CV в день"
2. Убедитесь, что сейчас рабочие часы
3. Проверьте логи на наличие ошибок

### Браузер не закрывается

1. Включите "Фоновый режим" в настройках
2. Перезапустите агента
3. Проверьте, что нет активных задач

## 💡 Рекомендации

### Оптимальные настройки

- **Интервал**: 2-3 часа (не слишком часто)
- **Максимум CV**: 10-20 в день (не спамить)
- **Рабочие часы**: 09:00-18:00 (рабочее время)
- **Фоновый режим**: включен (не мешает работе)

### Мониторинг

- Регулярно проверяйте статистику
- Следите за логами активности
- При необходимости корректируйте настройки
