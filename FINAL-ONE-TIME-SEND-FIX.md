#### 3. ✅ Добавлен реальный мониторинг прогресса

**Было:**
- Проверка каждые 15 секунд
- Показывает сразу "100 готово"
- Нет реального отслеживания

**Стало:**
- Проверка каждые 5 секунд (в 3 раза чаще!)
- Отслеживает прогресс ТЕКУЩЕЙ сессии
- Плавное обновление: 0/100 → 5/100 → 10/100 → ... → 100/100

**Логика:**
```typescript
// Запоминаем начальное количество
sessionStartCount = serverStats.totalSent; // Например, 250

// При каждой проверке (каждые 5 сек):
currentProgress = totalSent - sessionStartCount;
// 251 - 250 = 1 CV
// 252 - 250 = 2 CV
// ...
// 350 - 250 = 100 CV

// Обновляем UI:
setAutoApplyProgress(`Отправка CV: ${currentProgress} / 100`);
```
