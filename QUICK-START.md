# ⚡ Быстрый старт - Автоотправка CV

## 🚀 Запуск за 30 секунд

```bash
# 1. Запустить серверы
bash START-SERVERS.sh

# 2. Открыть в браузере
http://10.211.55.5:5004
```

## ⚙️ Настройка автоотправки

1. **Перейти:** Автоагенты → ⚙️ Настройки
2. **Установить:**
   - Максимум CV: **20**
   - Интервал: **3 часа**
   - Фоновый режим: **✓ Включен**
3. **Сохранить** → Увидите toast

## ▶️ Запуск автоотправки

1. Нажмите: **▶️ Автоотправка CV**
2. Откройте консоль: **F12**
3. Смотрите прогресс

## 📊 Проверка статуса

### В консоли браузера (F12):

```javascript
// Проверить настройки
JSON.parse(localStorage.getItem('cvflow_lucru_config')).settings

// Проверить sessionId
localStorage.getItem('cvflow_lucru_session')
```

### В терминале backend:

```
📊 ПРОГРЕСС: 5 / 20 вакансий
   ✅ Отправлено: 3 | ⏭️ Пропущено: 2
```

```bash
~/.cache/ms-playwright/chromium-1194/chrome-linux/chrome --version

```

## ❓ Проблемы?

- **Останавливается после пары вакансий** → Перезапустите backend
- **Настройки не сохраняются** → Проверьте localStorage (F12)
- **Агент включается-выключается** → Удалите cookies и войдите заново
- **Не видно прогресса** → Откройте консоль (F12)
