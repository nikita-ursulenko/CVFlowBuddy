# 🔧 Критическое исправление: Поддержка Lazy Loading

## Проблема

Агент обрабатывал **только 16 вакансий** вместо 100, потому что:

1. **Lucru.md использует Lazy Loading** - вакансии подгружаются только при прокрутке
2. **Агент не прокручивал страницу** - видел только первую порцию вакансий
3. **Результат:** Даже при настройке 100 CV → отправлялось только 4-16

### Пример из логов:

```
📋 Найдено вакансий: 16  ← Только первая порция!
📋 Итого вакансий для обработки: 16
...
📊 ИТОГИ:
   ✅ Отправлено: 4
   📋 Обработано: 4 из 16  ← Вместо 100!
```

## Решение

Добавлена **автоматическая прокрутка страницы** для загрузки всех вакансий.

### Алгоритм:

```
1. Открыть страницу вакансий
   ↓
2. Начальная прокрутка (0 вакансий)
   ↓
3. Прокрутить на 1000px вниз
   ↓
4. Подождать 1.5 сек (загрузка новых вакансий)
   ↓
5. Подсчитать вакансии
   ↓
6. Если количество изменилось → прокрутить еще
   ↓
7. Повторять пока:
   - Количество перестало расти ИЛИ
   - Загружено >= maxJobs вакансий ИЛИ
   - Выполнено 20 прокруток (лимит)
   ↓
8. Начать обработку всех загруженных вакансий
```

### Код:

**Файл:** `agent-server-simple.js` (строки 1145-1185)

```javascript
// КРИТИЧЕСКИ ВАЖНО: Прокручиваем страницу чтобы загрузить все вакансии (lazy loading)
console.log('📜 Прокручиваем страницу для загрузки всех вакансий...');

let previousVacancyCount = 0;
let currentVacancyCount = 0;
let scrollAttempts = 0;
const maxScrollAttempts = 20; // Максимум 20 прокруток

do {
  // Прокрутка вниз
  await this.page.evaluate(() => {
    window.scrollBy(0, 1000); // Прокрутка на 1000px вниз
  });
  await this.page.waitForTimeout(1500); // Ждем загрузки новых вакансий
  
  // Считаем текущее количество вакансий
  const currentRows = await this.page.$$('li.vacancyRow');
  currentVacancyCount = currentRows.length;
  
  console.log(`   📜 Прокрутка ${scrollAttempts + 1}: найдено ${currentVacancyCount} вакансий`);
  
  scrollAttempts++;
  
  // Если количество не изменилось - больше вакансий нет
  if (currentVacancyCount === previousVacancyCount) {
    console.log(`   ✅ Достигнут конец списка (вакансий больше нет)`);
    break;
  }
  
  previousVacancyCount = currentVacancyCount;
  
  // Если достаточно вакансий для обработки - останавливаем прокрутку
  if (currentVacancyCount >= maxJobs) {
    console.log(`   ✅ Загружено достаточно вакансий: ${currentVacancyCount} >= ${maxJobs}`);
    break;
  }
  
} while (scrollAttempts < maxScrollAttempts);

console.log(`📋 Всего загружено вакансий после прокрутки: ${currentVacancyCount}\n`);
```

## Как это работает

### Визуализация процесса:

```
Страница загружена
   ↓
📜 Прокрутка 1: найдено 16 вакансий
   ↓ (scroll 1000px, wait 1.5s)
📜 Прокрутка 2: найдено 32 вакансии
   ↓ (scroll 1000px, wait 1.5s)
📜 Прокрутка 3: найдено 48 вакансий
   ↓ (scroll 1000px, wait 1.5s)
📜 Прокрутка 4: найдено 64 вакансии
   ↓ (scroll 1000px, wait 1.5s)
📜 Прокрутка 5: найдено 80 вакансий
   ↓ (scroll 1000px, wait 1.5s)
📜 Прокрутка 6: найдено 96 вакансий
   ↓ (scroll 1000px, wait 1.5s)
📜 Прокрутка 7: найдено 112 вакансий
   ↓
✅ Загружено достаточно: 112 >= 100
   ↓
Начинаем обработку 100 из 112 вакансий
```

### Условия остановки прокрутки:

1. **Конец списка**
   ```javascript
   if (currentVacancyCount === previousVacancyCount) {
     // Количество не изменилось → конец списка
     break;
   }
   ```

2. **Достаточно вакансий**
   ```javascript
   if (currentVacancyCount >= maxJobs) {
     // Загружено 112 >= 100 → хватит
     break;
   }
   ```

3. **Лимит прокруток**
   ```javascript
   if (scrollAttempts >= 20) {
     // 20 прокруток выполнено → стоп
     break;
   }
   ```

## Параметры

| Параметр | Значение | Назначение |
|----------|----------|------------|
| Прокрутка | 1000px | Расстояние за одну прокрутку |
| Ожидание | 1.5 сек | Время загрузки новых вакансий |
| Макс прокруток | 20 | Лимит (защита от бесконечного цикла) |

### Расчет времени:

```
Для загрузки 100 вакансий:
- Примерно 7 прокруток (16 → 32 → 48 → 64 → 80 → 96 → 112)
- 7 прокруток × 1.5 сек = 10.5 секунд

Для загрузки всех вакансий (912):
- Примерно 20 прокруток (лимит)
- 20 × 1.5 сек = 30 секунд
```

## Защита от бесконечного цикла

### Максимум 20 прокруток:

```javascript
const maxScrollAttempts = 20;

if (scrollAttempts >= maxScrollAttempts) {
  // Остановка после 20 прокруток
  // Обработка того что загрузилось (обычно ~320 вакансий)
}
```

**Почему 20:** 
- 20 прокруток × 16 вакансий/прокрутка ≈ 320 вакансий
- Этого достаточно для 99% случаев
- Защищает от бесконечного цикла если сайт изменился
