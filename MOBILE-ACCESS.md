## 🔧 Ручное открытие портов

Если скрипт не помог, выполните вручную:

### Ubuntu с UFW:

```bash
# Открыть порт 5004
sudo ufw allow 5004/tcp

# Открыть порт 5050
sudo ufw allow 5050/tcp

# Перезагрузить правила
sudo ufw reload

# Проверить статус
sudo ufw status
```

### Ubuntu с iptables:

```bash
# Открыть порт 5004
sudo iptables -I INPUT -p tcp --dport 5004 -j ACCEPT

# Открыть порт 5050
sudo iptables -I INPUT -p tcp --dport 5050 -j ACCEPT

# Сохранить правила
sudo netfilter-persistent save
```

## ⚠️ Возможные проблемы

### Проблема 1: Телефон в другой сети
**Решение:** Подключите телефон к той же WiFi что и компьютер

### Проблема 2: Firewall блокирует порты
**Решение:** Выполните `sudo bash open-ports.sh`

### Проблема 3: Router блокирует соединения
**Решение:** Проверьте настройки роутера (AP Isolation должен быть выключен)

### Проблема 4: VPN активен на телефоне
**Решение:** Отключите VPN на телефоне
