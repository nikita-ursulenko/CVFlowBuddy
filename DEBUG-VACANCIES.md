
## Возможные причины:

1. **Структура изменилась** - Lucru.md обновили дизайн
2. **Вакансии загружаются через JavaScript** - нужно ждать дольше
3. **Элементы внутри Shadow DOM** - нужны другие методы доступа
4. **Lazy rendering** - элементы рендерятся постепенно
