## ⚠️ Важные замечания:

### 1. Сессия хранится в памяти

- После перезапуска `agent-server.js` нужно войти заново
- Сессия активна только пока работает сервер

### 2. Headless режим

- Сейчас браузер видимый (`headless: false`) для отладки
- Для продакшена измените в `agent-server.js`:
  ```javascript
  headless: true; // вместо false
  ```

### 3. Путь к CV файлу

- Для реальной отправки CV нужно указать путь к файлу
- Сейчас `cvFilePath: ''` - файл не загружается
- Можно добавить в конфигурацию агента

### 4. Порт сервера

- Сервер должен работать на порту **5050**
- Фронтенд настроен на этот порт
- Не меняйте порт без необходимости

## 🐛 Если что-то не работает:

### Проблема: "Сервер агента недоступен"

**Решение:** Проверьте, что `agent-server.js` запущен на порту 5050

### Проблема: "Сессия не найдена"

**Решение:** Перезапустите `agent-server.js` и войдите заново

### Проблема: "Найдено 0 вакансий"

**Решение:** Проверьте интернет-соединение и доступность lucru.md

### Проблема: TypeScript ошибки

**Решение:** Все ошибки уже исправлены, перезагрузите VS Code
