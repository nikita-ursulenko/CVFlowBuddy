## 🔍 Диагностика

### Если всё равно обрабатывается мало вакансий:

**Проверьте логи:**
1. Сколько вакансий загружено после прокрутки?
2. Почему остановилась прокрутка?
3. Какие вакансии пропускаются и почему?

**Команды для проверки:**
```bash
# Логи сервера в реальном времени:
tail -f /dev/stdout (открыть терминал где запущен agent-server)

# Или в консоли браузера (F12)
```

### Если вакансии пропускаются:

Смотрите debug информацию:
```
⏭️  Пропускаем: нет названия (debug: Microsoft Dynamics Engineer ...)
```

Это означает что элемент найден, но название не извлечено. 
Нужно добавить еще селекторы для этой вакансии.
